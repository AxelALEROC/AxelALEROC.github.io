Tecnologías y lenguajes

Hola, Soy Axel.

Soy un desarrollador backend de aplicaciones web y escritorio, programo desde 2021; para la parte Backend he trabajado en lenguajes como C#, Java y Python. Especialmente trabajando directamente con C# con entorno .NET y ASP.NET para la creación de proyectos usando Frameworks como EntityFramework y ADO.NET para manejar ORM´s para servicios RESTful y APIS REST. Para la parte Fronted he trabajado con HTML5, CSS3, JavaScript, React JS y Blazor; Por su parte para bases de datos he trabajado con SQL SERVER y OracleDB.

Aplicaciones web y escritorio

Enfocado principlamente para la creación de aplicaciones web y de escritorio con uso de entorno .NET y ASP.NET para páginas web para servicios API REST con arquitectura MVC.

Aplicaciones en consola

Desarrollo de aplicaciones de consola en Java para abordar problemas lógicos y desafíos de programación Utilización de programación orientada a objetos (POO) para el modelado de entidades y el manejo de transacciones.

Analisis de datos y Web Scraping

Desarrollo de web scraping en Python para recopilar datos relevantes de sitios web públicos. Implementación de bibliotecas y herramientas de análisis de datos como pandas, NumPy y matplotlib.
